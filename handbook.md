# 学院用户使用手册（当前功能版）

本手册面向学院用户，介绍登录、上传与解析预览、历史记录查看、常见问题与自检清单等操作。管理员功能（汇总统计、学院管理、测试样例、汇总输出）不在本手册范围，但文末提供如何查看目标达成的建议。

## 界面与角色
- 学院用户左侧菜单包含：上传数据、历史记录

## 登录、退出与切换账号
- 登录：输入用户名与密码后点击“登录”
  - 用户名示例：`user_economy`、`user_finance`、`user_intl`、`user_west`、`user_tax`、`user_mgmt`
  - 初始密码：由联络员单独发放。管理员可重置密码
- 登录状态提示：
  - 错误：显示“用户名或密码错误”
  - 未登录：显示“请输入用户名和密码”
- 退出登录：侧栏点击“退出登录”
- 切换账号：侧栏点击“切换账号”清理会话并返回登录页

## 上传数据（问答对 / 习题库）
1. 菜单选择“上传数据”
2. 点击“下载模板”页签，下载并参考模板填充数据：
   - 问答对模板（CSV）：包含 `question`、`answer`
   - 习题库模板（CSV）：包含 `type`（可选）、`stem`、`options`（选择题可选）、`answer`、`knowledge`（可选）、`analysis`（可选）
3. 切换至“文件上传与解析”页签：
   - 选择上传类型：问答对 或 习题库
   - 如为习题库，选择题型（或选择“自动识别”）
   - 上传 `xlsx`/`xls`/`csv` 文件，完成后系统自动解析
4. 解析成功后会显示概览与预览（见下一节）
5. 解析结果自动保存为 CSV（按中文学院名与日期归档），便于后续历史查看与汇总

## 解析预览与差异化抽检
解析成功后展示三个预览页签，用于快速自检数据质量。
- 随机抽检（20）：随机展示最多 20 条，便于快速抽查质量与格式
- 顺序浏览：分页浏览、关键词搜索；可输入“答案为空”快速定位空答案条目
- 类型抽检（差异化校验）：按题型进行专属校验与统计
  - 问答对：总条目数、答案为空数量、问题长度分布
  - 判断题：答案分布柱状图（观察是否均衡）
  - 选择题：校验选项完整性与答案一致比例
    - `options` 列建议为多行文本，每行形如：`A: 选项A`、`B: 选项B`（可含 C/D…）
    - `answer` 应为选项字母（如 `A`/`B`）
  - 填空题：答案非空比例
  - 简答/论述/案例分析：`analysis` 解析非空比例

## 历史记录查看
- 菜单选择“历史记录”
- 按日期列出上传记录，点击每条可展开预览与类型抽检
- 历史记录仅展示本学院数据；如需删除或更改请联系管理员

## 模板与字段规范（建议）
- 问答对：
  - 必需列：`question`、`answer`
  - 建议：问题明确、答案非空；避免冗余或含个人敏感信息
- 习题库：
  - 必需列：`stem`、`answer`
  - 推荐列：`options`（选择题）、`knowledge`（知识点）、`analysis`（解析）
  - 选择题 `options` 建议：多行文本，每行 `字母 + 冒号 + 空格 + 内容`；`answer` 使用选项字母

## 常见问题与自检清单
- 上传后显示“未识别到有效数据”：
  - 检查文件类型是否为 `csv/xlsx/xls`
  - 检查是否包含必需列（问答对：`question`、`answer`；习题库：`stem`、`answer`）
- 选择题一致性比例低：
  - 检查 `options` 行格式是否规范（至少包含 `A:`、`B:`）
  - 检查 `answer` 是否在选项字母集合中
- Excel 多 Sheet 未识别：
  - 检查目标 Sheet 是否包含必需列；概览会显示识别到的 Sheet 名称与列名
- 编码乱码：
  - CSV 建议使用 UTF-8；若为 GBK/ANSI，建议转换为 UTF-8 再上传

## 如何查看目标与达标（学院用户）
- 目标设置与达标状态由管理员在“学院管理”进行维护与查看，学院用户默认不可见
- 建议与管理员确认以下信息：
  - 本学院问答对与习题总目标数量，以及各题型目标数量
  - 目前的达标状态（达标/未设定/未达标）与剩余量
- 自助估算方法：
  - 在“历史记录”中展开各上传文件的预览，累计问答对与习题条目数，作为当前进度参考
  - 使用“类型抽检”看到各题型的分布情况（选择题、判断题等），帮助判断各题型是否接近目标
- 平台支持按学院汇总统计与目标对比
